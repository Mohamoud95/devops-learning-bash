# Script 72: Create a script to rename a file or directory.

## Purpose
This script automates the process of: **Create a script to rename a file or directory.**

## Script Code (`72-create-a-script-to-rename.sh`)
```bash
#!/bin/bash

# Create a script to rename a file or directory.

OLD_NAME="old_
fi
le.txt"NEW_NAME="new_
fi
le.txt"mv $OLD_NAME $NEW_NAME
echo "File renamed to $NEW_NAME."
```

This script uses standard Linux utilities to achieve its goal. It processes the required input and executes the logic step-by-step to fulfill the requirements of Bash Problem 72.

- `bash` (script execution)
- Utilities utilized: `echo`, `mv`
